# LearnCare Chart MVP

Web MVP project for the Ministry of Education public data utilization competition.

## Branches

- `main`: stable release branch
- `develop`: integration branch
- `feature/fe-*`: frontend feature branches
- `feature/be-*`: backend feature branches
- `fix/*`: bug fix branches
- `chore/*`: setup/config branches
- `docs/*`: documentation branches
- `ai/codex-*`, `ai/claude-*`: AI-generated work branches

## Basic Workflow

1. Create a branch from `develop`.
2. Commit focused changes.
3. Open a pull request into `develop`.
4. Request review before merging.
5. Promote `develop` to `main` only for stable releases.

See [docs/GIT_WORKFLOW.md](docs/GIT_WORKFLOW.md) for the full team workflow.
